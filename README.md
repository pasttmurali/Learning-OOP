# Learning OOP

This repository contains my Java programming practice work, including basic Java concepts before Object-Oriented Programming and Object-Oriented Programming examples.

## Repository Structure

```text
Learning-OOP/
│
├── OOP/
```

## Contents

### 1. Java Learning Before OOP

This folder includes beginner-level Java programs such as:

* Variables and data types
* Input and output
* Conditional statements
* Loops
* Arrays
* Strings
* Methods
* Basic problem-solving programs
* Past paper practice questions

### 2. OOP

This folder includes Object-Oriented Programming concepts such as:

* Classes and objects
* Constructors

* Interfaces
* Java OOP practice programs

The purpose of this repository is to organize my Java learning progress from basic programming concepts to Object-Oriented Programming.

## How to Run Java Files

Open the terminal inside the folder where the `.java` file is located and run:

```bash
javac FileName.java
java FileName
```

Example:

```bash
javac Student.java
java Student
```
